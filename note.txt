1. git clone: kéo dự án trên server về máy tính.
2. git init: Khởi tạo thư mục git local
3. git status: kiểm tra status của file
4. git add . : đưa tất cả các file có sự thay đổi so với file trước vào vùng staging
5. git add [tên file]: đưa 1 file có sự thay đổi vào vùng staging
6. git rm --cached [Tên file]: đưa 1 file từ staging quay trở lại working directory
7. git commit: đưa vào local respon
